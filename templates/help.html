Help

Search:

Perform a text-based search of all genes and coding sequences (CDSs) in the database. The search returns all features containing the search term in any field. e.g. searching for 'leuS' returns genes and CDSs in the selected species with the name 'leuS' or with 'leuS' mentioned in any other field. Similarly, searching for 'synthetase' returns any features with a synthetase as a gene product or with 'synthetase' mentioned in any other field. Selecting a particular gene or CDS allows the user to view further details of that feature and to scroll through the genome to view neighbouring coding sequences. In the case of CDSs, orthologues in other genomes are displayed below the genome of interest.

Genomes:

Displays details of all genomes currently in the database.

MultiFun:

View genes/CDSs based on the E.coli 'MultiFun' (multi-functional) classification system. The leftmost column shows all E.coli CDSs for the selected classification (e.g. '1 Metabolism' or '7.4 Outer membrane'). The columns on the right show the orthologue CDSs in other replicons (either chromosome or plasmid of another species). By default, only the E.coli CDSss which have orthologues are displayed, but selecting the 'Show all E.coli genes' checkbox displays all CDSs.
