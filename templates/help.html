Help

Search:

Perform a text-based search of all genes and coding sequences in the database. The search returns all features containing the search term...
Selecting a particular gene allows you to view further details about the coding sequence, including its orthologues in other genomes, and to scroll through the genome to view neighbouring coding sequences.

Genomes:

Displays details of all genomes currently in the database.

MultiFun:

View genes and coding sequences based on the E.coli 'MultiFun' (multi-functional) classification system.
